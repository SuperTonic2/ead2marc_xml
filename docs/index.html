Defaults are IUL conventions (035 prefix MC, MARC code Inu-MuID, cataloging source IUL, holdings code IULA). Edit if your institution uses different codes; your values are saved in this browser and used for future conversions. The 035 row appears only when "collection" is selected as the hierarchy level, since 035 is only emitted for collection-level records.
