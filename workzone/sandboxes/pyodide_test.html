Pyodide Feasibility Check

What this is: a self-running test of whether your EAD2MARC_XML conversion logic can run entirely in a web browser via Pyodide (Python compiled to WebAssembly).

What to do:

  1. Wait for all tests to complete (~30-90 seconds; Pyodide is ~10MB so first load is slow)
  2. Once the yellow "Summary" box appears, click Copy results to clipboard
  3. Paste the result back to Claude

Open in: Firefox is best for this test (more permissive about local file CORS). Chrome will work but may show stricter CORS failures. If you can test in both, that's even more useful.
